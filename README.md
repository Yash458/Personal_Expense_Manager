# Personal Expense Manager Android Application 
### •	Designed and developed an android mobile application which allows user to add and maintain their day-to-day expenses
### • Application has functionalities like unique login id which lets user access from different devices as well also it has chart analysis for daily, weekly and monthly expenses.
### • Coding language - Java
### • Data storage - Firebase
### • IDE - Android studio
